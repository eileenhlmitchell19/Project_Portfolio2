Employee Tracking

This application allows users to view tables for a companies employees, roles, and departments. The application also allows users to manipulate data by adding additional members to the team, add new employee roles or departments. Once updated, the user can then view their input into the table. All information can be accessed via the terminal by typing "node index.js" and following prompts.
